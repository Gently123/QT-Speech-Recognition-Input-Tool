# Webserver 项目说明

## 安装和使用
- **依赖环境**：需要 Qt 5.63 或更高版本，以及对应版本的openssl（其实QT安装目录里可以找到那两个dll文件）。
- **编译步骤**：
  1. 打开 Qt Creator 或命令行。
  2. 加载 `Webserver.pro` 项目文件。
  3. 构建项目（Debug 或 Release 模式）。
  4. 运行生成的可执行文件。
- **使用说明**：启动应用后，使用全局热键 `Ctrl+Alt+L` 开始录音，`Ctrl+Alt+D` 结束录音。勾选打字，识别结果将自动输出到当前光标位置。

## 客户端核心功能
- **打字输出模式**：通过将识别结果复制到剪切板并模拟 `Ctrl+D` 自动输出到当前光标位置，实现“语音即打字”。
- **全局热键控制**：使用 `Ctrl+Alt+L/D` 控制录音的开始与结束，支持后台无干扰工作。
- **小窗口模式**：提供悬浮麦克风图标，实时显示音量并支持拖拽操作。
- **多模式识别**：支持离线、在线、2pass 混合识别，还可识别语音文件，通过连接或断开切换模式。

## 技术实现
- **异步音频处理**：基于 Qt 信号槽机制，实现实时录音、编码与传输，确保 UI 线程非阻塞。
- **WebSocket 全双工通信**：基于 TCP/IP 通信协议，支持 SSL/TLS 加密。
- **系统集成**：利用 Windows API 实现键盘模拟、系统托盘与全局热键功能。
- **智能音频处理**：支持 WAV 文件头解析、音频分块传输与音量可视化。

## 用户体验优化
- **无缝切换**：自由切换正常模式与打字模式。
- **实时反馈**：通过 7 级麦克风动画实时响应音量变化。
- **后台运行**：支持最小化到托盘并在后台录音。
